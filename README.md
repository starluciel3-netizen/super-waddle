# super-waddle
Un espacio para intentarlo, fallarlo, aprender y avanzar.
Codigo sencillo, ideas claras y progreso constante.

## ¿Què es esto?
**super-waddle** es un proyecto personal enfocado en aprender Java desde cero, 
entendiendo los fundamentos y construyendo paso a paso sin atajos.

## Objetivo
-Aprender Java de forma pràctica
-Crear hàbitos reales de desarrollo
-Usar GitHub como portafolio, no como bodega

## Tecnologias
-Java (Eclipse Temurin JDK)
-Git & GitHub
-VS Code / IDE de Java
